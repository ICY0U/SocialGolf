# Quick Start Guide: Candle System Setup

## ?? **Getting Started**

Your candle system is now fully implemented and ready to use! Here's how to get started immediately.

## ? **Instant Testing**

### 1. **In-Game Console Commands**
Open console with **`** key and try these commands:

```
SpawnTestCandle        // Spawns a lit candle in front of you
LightAllCandles        // Lights all candles in the level  
ExtinguishAllCandles   // Puts out all candles
ToggleAllCandles       // Toggles all candles on/off
```

### 2. **Quick Level Setup**
1. **Open your level** in Unreal Editor
2. **Search** for "SGCandle" or "SGAdvancedCandle" in the Content Browser
3. **Drag** a candle into your level
4. **Set** `Start Lit = true` in the Details panel
5. **Play** and test interaction with E key

## ?? **Performance Tips**

### **For Better Performance:**
- Use **`LightMode = Steady`** for background candles (no flickering)
- Limit **Magical candles** to special areas only  
- Use **Basic SGCandle** for simple needs
- Use **Advanced SGCandle** for hero lighting

### **For Better Visuals:**
- Mix different candle types in the same room
- Use **fill lighting** with other light sources
- Combine with particle effects for atmosphere
- Adjust `FlickerIntensity` based on candle size

## ?? **Quick Troubleshooting**

### **Candle won't light:**
- Check you're looking directly at it
- Verify `bCanBeExtinguished = true`
- Try console command `LightAllCandles`

### **No focus effect:**
- Ensure `SGFocusableComponent` is added
- Check `bEnableFocusEffect = true`
- Verify interaction component settings

### **Performance issues:**  
- Use `stat fps` to monitor framerate
- Switch background candles to Steady mode
- Reduce magical candles count
